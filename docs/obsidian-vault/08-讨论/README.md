# 讨论区

这里专门用于多 Agent / 多角色的讨论、评审、异议和结论沉淀。

## 使用约定

- 讨论统一放在 `08-讨论/`
- 先按主题域拆目录，再按具体议题建笔记
- 笔记命名统一用 `主题-xxx.md`
- 如果是围绕某一份现有方案展开，正文第一段直接链接原文档
