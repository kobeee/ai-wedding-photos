# V5 多维矩阵评审协议

> 评审时间: 2026-04-03
> Harness 版本: V5 Matrix
> 评审员: Codex (GPT-5.4)

## 评审目标

对 24 个测试 case 的三轮法生成结果进行系统化评审，定位跨维度弱点。

## 评审维度（每张图 3 项，每项 0.0-1.0）

| 维度 | 含义 | 关注点 |
|------|------|--------|
| **identity_match** | 面部与参考图一致性 | 五官形态、肤色、发型、体型 |
| **brief_alignment** | 场景与创意简报匹配度 | 故事情绪、服装、环境、构图意图 |
| **aesthetic_score** | 摄影美学水准 | 光影、色调、构图、情感冲击力 |

## 评审标准

```
A+  (0.95+): 顶级作品，可直接交付
A   (0.90-0.94): 优秀，微瑕不影响交付
A-  (0.85-0.89): 良好，有明确可改进点
B+  (0.80-0.84): 合格，���在明显但不致命的问题
B   (0.75-0.79): 边缘，需要修复后才能交付
C   (0.70-0.74): 不合格，需要重跑
D   (<0.70):    严重失败
```

## 硬失败条件（一票否决）

- 多手指/缺手指/肢体变形
- diptych/collage/split-screen 多联画
- 身体穿模（衣服/道具）
- 面部严重变形
- 出现文字/水印/不可能物体
