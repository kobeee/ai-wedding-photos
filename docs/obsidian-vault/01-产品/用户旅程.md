# 用户旅程

> 基于 [[PRD分析]] 中定义的 5 步操作路径，展开每一步的完整交互链路。

---

## 总览流程图

```
建立档案 → AI试妆 → 挑选套餐 → 沉浸式等待 → 审片与交付
 /upload    /makeup   /package    /waiting     /review
```

每一步对应前端独立页面，通过 [[系统架构#前端页面结构]] 中定义的路由串联。

---

## Step 1：建立档案（/upload）

### 前端交互
- 用户进入上传页，分别上传男方/女方各 5 张日常照
- 拖拽或点击上传，支持批量选择
- 每张照片展示缩略图预览，可单独删除
- 上传进度条实时反馈
- 全部上传完成后，"下一步"按钮激活

### 后端处理
- 接收文件 → 调用 [[API设计#POST /api/upload]]
- 文件存储到临时目录
- 提取 Face-ID Embeddings（特征向量）
- 返回上传确认 + 档案 ID

### AI 管线触发
- 无，此步纯数据采集

### 设计要点
- 参考 [[设计系统]] 的深色主题，上传区域使用虚线边框
- 进度条使用金色（`--accent-gold`）

---

## Step 2：AI试妆（/makeup）

### 前端交互
- 展示 3 种妆造风格预览：素颜、精致妆、骨相微调
- 左右滑动或点击切换
- 选中后高亮金色边框
- 确认选择后进入下一步

### 后端处理
- 调用 [[API设计#POST /api/makeup/generate]]
- 传入档案 ID + 妆造类型
- 后端调用 [[Nano-Banana-Pro]] 生成 3 张试妆效果图
- 返回 3 张预览图 URL

### AI 管线触发
- [[Nano-Banana-Pro]] 文生图/图生图混合模式
- 输入：用户面部照 + 妆造 Prompt
- 输出：3 张 512px 预览图

### 设计要点
- 大图对比展示，突出"变化感"
- 选择交互要流畅，减少决策压力

---

## Step 3：挑选套餐（/package）

### 前端交互
- 卡片式瀑布流浏览，类似小红书体验
- 每张卡片展示：样片、风格名、简短描述
- 无文本输入，纯视觉选择
- 点击卡片展开大图预览
- 选中套餐后确认

### 后端处理
- 前端从静态配置加载套餐数据（Phase 1）
- 选择结果发送给后端，关联到档案 ID
- LLM 将套餐映射为 Camera Schema

### AI 管线触发
- LLM 中枢调度层启动
- 套餐 → Camera Schema → Prompt 指令编译

### 设计要点
- 参见 [[设计规范]] 的卡片设计规范
- 套餐卡片使用 16px 圆角，hover 时金色描边

---

## Step 4：沉浸式等待（/waiting）

### 前端交互
- 全屏居中等待界面
- 进度条 + 当前步骤文字提示
- 文案安抚（"AI 摄影师正在为您精心布光..."）
- 预计等待 30-60 秒
- 轮询后端状态接口

### 后端处理
- 前端轮询 [[API设计#GET /api/generate/{task_id}/status]]
- 后端返回进度百分比 + 当前阶段描述
- 生成完成后返回 `completed` 状态

### AI 管线触发
- [[Nano-Banana-Pro]] 渲染 4K 底图
- [[VLM质检管线]] 启动视觉反思循环
- 如有问题，触发 [[双模型协作]] 修复流程
- 最多 3 次循环直至合格

### 设计要点
- 参见 [[设计系统]] 等待页面设计
- 全屏居中，背景微动效
- 进度条使用金色渐变
- 情绪文案轮播

---

## Step 5：审片与交付（/review）

### 前端交互
- 左大图 + 右操作面板布局
- 大图区域支持缩放查看细节
- 操作面板：下载、换装、重拍
- 下载选项：标清（免费水印）/ 4K / 8K
- 付费墙在下载高清时触发

### 后端处理
- 调用 [[API设计#GET /api/generate/{task_id}/result]]
- 返回多分辨率图片 URL
- 付费验证后返回无水印版本

### AI 管线触发
- 魔法笔刷微调（按需）
- 8K 超分放大（按需）

### 设计要点
- 大图震撼呈现 = 关键转化节点
- 参见 [[页面清单#审片与交付]] 的布局细节
- 下载按钮使用金色 CTA 样式

## 关键指标

| 指标 | 目标 |
|------|------|
| 上传→试妆转化率 | > 80% |
| 套餐选择时间 | < 60 秒 |
| 等待放弃率 | < 10% |
| 审片→下载转化率 | > 50% |
| 免费→付费转化率 | > 15% |
