## 部署架构

> 详见 [[部署方案]]

```
用户 → wedding.escapemobius.cc
     → VPS Nginx (65.75.220.11)
       → Docker Compose
         ├── frontend (Nginx:3080 → 80)
         └── backend  (Uvicorn:8000, 仅内部)
```

### Docker 端口规划

| 服务 | 容器端口 | 主机端口 | 对外 |
|------|----------|----------|------|
| 前端 | 80 | 3080 | 是 |
| 后端 | 8000 | 8000 | 仅内部 |
| 数据库 | - | - | 禁止暴露 |

> Docker 配置文件详见 [[Docker配置]]，Nginx 配置见 [[Nginx配置]]

## 关键设计决策

1. **前后端分离** - 前端通过 Nginx 反代 `/api/` 到后端
2. **SPA 路由** - Nginx fallback 到 `index.html`
3. **去 AI 化** - 全流程隐藏技术参数，参见 [[设计规范]]
4. **移动端优先** - 后续迭代，Phase 1 以桌面端为主

---

## 安全红线

- 数据库端口**严禁暴露**公网
- 用户面部数据生成后 24 小时内自动销毁
- 所有 API 接口必须鉴权
