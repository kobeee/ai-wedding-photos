## AI 模型集成

### Nano Banana Pro 调用

参见 [[Nano-Banana-Pro]] 的 API 文档。

```python
import httpx

async def call_nano_generate(prompt: str, images: list[bytes] = None):
    async with httpx.AsyncClient() as client:
        payload = {
            "contents": [{
                "parts": [{"text": prompt}]
            }],
            "generationConfig": {
                "responseModalities": ["IMAGE"],
                "imageConfig": {"aspectRatio": "3:4", "imageSize": "1024"}
            }
        }
        if images:
            for img in images:
                payload["contents"][0]["parts"].append({
                    "inline_data": {
                        "mime_type": "image/jpeg",
                        "data": base64.b64encode(img).decode()
                    }
                })
        resp = await client.post(NANO_ENDPOINT, json=payload, headers=headers)
        return resp.json()
```

from openai import AsyncOpenAI

client = AsyncOpenAI(api_key=API_KEY, base_url="https://api.laozhang.ai/v1")

async def call_gpt_edit(image_path: str, prompt: str, mask_path: str = None):
    kwargs = {
        "model": "gpt-image-1",
        "image": open(image_path, "rb"),
        "prompt": prompt,
        "n": 1,
        "size": "1536x1024"
    }
    if mask_path:
        kwargs["mask"] = open(mask_path, "rb")
    return await client.images.edit(**kwargs)

## 异步任务处理

婚纱照生成是耗时操作（30-60s），需要异步处理：

1. 用户提交生成请求 → 返回 `task_id`
2. 后台启动异步任务（asyncio.create_task 或 Celery）
3. 前端轮询状态接口
4. 任务完成后存储结果

Phase 1 使用 `asyncio.create_task`，后续按需引入 Celery/Redis。

---

## 安全考虑

- API Key 通过环境变量注入，不硬编码
- 用户上传文件大小限制（单文件 10MB）
- 文件类型校验（仅接受 jpg/png/webp）
- 面部数据 24 小时自动销毁
- CORS 配置仅允许指定域名
