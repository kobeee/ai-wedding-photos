# 设计系统

> 原"设计笔记.md"重建，完整记录项目的视觉设计语言。

---

## 设计语言定位

**深色奢华 + 金色点缀**，面向 95 后/00 后备婚群体。

关键词：高端、温度感、沉浸式、去 AI 化。

> 用户画像来源：[[PRD分析#目标用户]]

---

## 色彩系统（17 个 Token）

### 核心色值

| Token | 值 | 用途 |
|-------|------|------|
| `--bg-primary` | `#0A0A0A` | 页面底色 |
| `--bg-surface` | `#141414` | 卡片/区块背景 |
| `--bg-surface-hover` | `#1A1A18` | 卡片 hover |
| `--accent-gold` | `#C9A96E` | 金色 CTA / 强调 |
| `--accent-gold-hover` | `#D4B87E` | 金色 hover |
| `--accent-gold-dim` | `rgba(201,169,110,0.1)` | 金色底色 |
| `--text-primary` | `#F5F0EB` | 主标题 |
| `--text-secondary` | `#B8B0A8` | 正文 |
| `--text-muted` | `#6B6560` | 辅助文字 |
| `--border-subtle` | `#2A2520` | 分割线 |
| `--gradient-gold` | `linear-gradient(135deg, #C9A96E, #E8D5A3)` | 金色渐变 |

### 语义色

| Token | 值 | 用途 |
|-------|------|------|
| `--success` | `#4CAF50` | 成功状态 |
| `--error` | `#F44336` | 错误状态 |
| `--warning` | `#FF9800` | 警告状态 |

这些 CSS Variables 定义在 `src/frontend/src/index.css` 的 `:root` 中。参见 [[系统架构#前端源码结构]]。

## 字体系统

| 属性 | 值 | 说明 |
|------|------|------|
| 字体族 | Inter | 唯一字体 |
| 标题 weight | 200 | 极细，高级感 |
| 正文 weight | 300-400 | 舒适阅读 |
| 按钮 weight | 500-700 | 醒目 |
| letter-spacing | 0.05-0.2em | 大量使用，提升高级感 |

### 字号规范

| 用途 | 大小 | weight | letter-spacing |
|------|------|--------|---------------|
| Hero 主标题 | 64px | 200 | 0.15em |
| 页面标题 | 36-48px | 200 | 0.1em |
| 区块标题 | 24-28px | 300 | 0.08em |
| 正文 | 16-18px | 300 | 0.02em |
| 辅助文字 | 13-14px | 400 | 0.05em |
| 按钮 | 14-16px | 500 | 0.15em |

## 间距系统

|------|------|----------|
| xs | 8px | 元素内间距 |
| sm | 16px | 相关元素间距 |
| md | 24px | 区块内间距 |
| lg | 32px | 区块间间距 |
| xl | 48px | 大区块间间距 |
| 2xl | 64px | 页面级间距 |

---

## 阴影与特效

- 卡片阴影：`0 4px 24px rgba(0,0,0,0.3)`
- 金色光晕：`0 0 20px rgba(201,169,110,0.15)`
- 背景纹理：微弱噪点叠加（opacity 0.03）
- 分割线：`1px solid var(--border-subtle)`

## 动画规范

| 场景 | 属性 | 时长 | 缓动 |
|------|------|------|------|
| Hover 过渡 | all | 0.3s | ease |
| 页面切换 | opacity + transform | 0.5s | ease-in-out |
| 进度条 | width | 0.5s | linear |
| 等待动画 | 脉冲 | 2s | ease-in-out infinite |
