# 2026-04-08 研发日志

## 端到端测试（失败）
- 用 5 张测试图片 + 冰岛场景 + 免费体验套餐跑完整管线
- **结果：0/3 张成片，全部失败**
- 耗时 ~22 分钟，大量 API 调用浪费

### 失败根因
1. **VPS 跑的是旧代码**（四轮法 + 双轨制），V14 两轮法代码没部署
2. **冰岛场景 prompt 偏远景**，模型生成人物极小（face_area_ratio 0~0.01，阈值 0.035）
3. **重试死循环**：prompt 不变 → 结果不变 → 3 次验证全失败 → 换 attempt 继续失败
4. **API 限流**：大量重试触发 429/503

### 发现的具体问题
- `nano_reference_max_images: 4`，第 5 张参考图被丢弃（V14 说的 14 图并发没实现）
- VLM 质检耗时 37~66 秒/次，太慢
- editorial_pipeline.py 还在跑 r3_image/r4_image（旧四轮逻辑）
